# 🤖 AI EXPO 1.0 – Machine Learning Projects Portfolio

🚀 A modern web-based dashboard that showcases multiple deployed Machine Learning projects in one place.  
This project was built for **AI EXPO 1.0** to demonstrate real-world ML applications across different domains.

---

## 🌟 Overview

This portfolio brings together **5 deployed Machine Learning projects** with an interactive and visually appealing UI.  
Each project demonstrates practical implementation of ML techniques including:

- 🧠 Supervised Learning
- 🖼️ Computer Vision (CNN)
- 📝 Natural Language Processing (NLP)
- 🎙️ Audio Signal Processing
- 🎭 Deep Learning for Emotion Detection

---
